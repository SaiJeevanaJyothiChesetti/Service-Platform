## 🎓 How to Use Each Feature

### Become a Provider
1. Login as customer
2. Click any service card
3. Click "Register as Provider"
4. Fill provider details
5. Click "Save Provider"
6. ✅ Profile appears in service list

### Book a Service
1. Login as customer
2. Click service from dashboard
3. Find provider
4. Click "Book This Provider"
5. ✅ Booking saved and shows in My Bookings

### Manage Profile
1. Click "My Profile" in sidebar
2. **For Customers**: Edit info only
3. **For Providers**: Edit info + see all services
4. Click "Save Changes"
5. ✅ Changes persist

### Create Service Profile
1. Click "Add Provider" in sidebar
2. Choose service type
3. Fill all details
4. Click "Save Provider"
5. ✅ Profile saved and shown in service page + profile page
